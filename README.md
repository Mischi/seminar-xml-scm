## Installationsanleitung

	- install [Node](http://nodejs.org/)
	- git clone https://github.com/Mischi/seminar-xml-scm.git
	- cd seminar-xml-scm
	- npm install -g grunt-cli
	- npm install

## Projekt starten

	- cd seminar-xml-scm
	- grunt
